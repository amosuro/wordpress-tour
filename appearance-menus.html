1. Managing Menus

    This section allows you to manage the menus featured on the site. In most cases there will be at least one menu here, but sometimes you can have access to the menus featured in a footer for example

  2. Be careful when editing the website's menus, because depending on the design of your website you may break the layout depending on what you change

  3. You can switch to edit different menus using this dropdown

  4. This left panel allows you to browse through all of the website's pages/posts/categories/links so that you can add them to the menu in the right panel

  5. In this panel you can see all of the menu items in the selected menu. Simply drag and drop to change the ordering!
